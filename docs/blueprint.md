# **App Name**: VersoraAI

## Core Features:

- User Authentication: Secure authentication system with Firebase Auth, supporting email/password login. Automatically redirects users to their role-specific dashboard after login.
- Role-Based Dashboards: Custom dashboards for students, instructors, and admins, each providing role-specific information and actions, tailored to their specific needs and responsibilities within the LMS.
- Student Learning Path: The dashboard presents students with personalized content for completing learning objectives, including a 'Continue Learning' section, 'My Courses', 'Upcoming Deadlines', 'Recent Feedback', and a CTA for their 'Evidence-of-Learning Portfolio'.
- Course Management: Enable instructors to create, manage, and publish course content with the content studio, utilizing a two-panel layout for efficient navigation and content management.
- AI Writing Feedback: Integrate AI to provide students with writing feedback within the Writing Center, offering clarity, structure, and citation suggestions via a sidebar AI assistant.
- AI-Powered Rubric Generation: Allow instructors to use the AI tool to generate grading rubrics within the Assignment & Rubric Studio, based on assignment prompts and learning objectives, with options to edit, approve, or reject the generated rubrics.
- Admin Cost Management Dashboard: Provide admins with insights into system costs, including AI usage, with visualizations like donut charts and line graphs, presenting data on total monthly cost, cost per student, and AI usage cost, broken down by service and course/department.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trust and intellect.
- Background color: Very light gray (#F5F5F5), near white to provide a clean and uncluttered base for the UI.
- Accent color: Vivid orange (#FF9800) for CTAs, providing a strong contrast and a sense of action.
- Headline font: 'Space Grotesk', sans-serif, lends a modern techy look. Body font: 'Inter', sans-serif, used for longer text.
- Use academic-themed icons, consistent with a modern learning environment.
- Card-based layout with a focus on clean design for academic use.
- Subtle animations on transitions and data updates to maintain user engagement.
